How It Works & Security

Encryption at Rest

All files are encrypted with AES-256-GCM before storage. Only encrypted (.enc) files are stored on the server.

Key Management

Each file gets a unique encryption key, stored separately. Keys are never sent with files unless requested.

Three Download Options

  • Auto-decrypt: Server decrypts automatically (convenient)
  • .enc + Key: Download encrypted file + key separately (most secure)
  • Local decryption: Use the provided Python tool for offline decryption
